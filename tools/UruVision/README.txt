UruVision:
	Real-time visualization of server activity displayed as a radial graph
	Inspired by gnuTellaVision:
		 http://www.sims.berkeley.edu/~rachna/gtv/
		(Does not use any of their Code!)

Requires:
	Python >= 2.2.1
	Tk for Python (included by default)

Usage:
	python UruVision.py ServerHostname [ServerPort] ServerPath
	-or-
	python UruVision.py
		(edit UruVision.ini to change all options)
Status:
	Unsupported
	
Known Issues:
	Cumulative memory leak. Don't leave it running all day.

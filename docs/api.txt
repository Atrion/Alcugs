**Apps**
--------
uru_* -> The Alcugs Uru Servers.
uruping -> Uru protocol Ping.
urucmd -> A basic Uru Protocol client.
urumsgtest -> Test big transmissions suit.

**Main**
--------
config -> Set some settings before compilation. (Debug level,enable dmalloc,...)
debug -> debugging interface
data_types -> Main Data types.
urustructs -> Some Uru structs.
license -> Alcugs License
prot -> The Uru Protocol
protocol -> Directly Associated with the Uru Protocol and the netcore.
urunet -> The Netcore, the heart of the servers.
stdebug -> The Debugging/Logging interface
settings -> Set some default config values values
version -> Servers version

**Util**
--------
conv_funs -> Urustring en/decoder, string filter, guid<->string, get_stime,ascii<->hex
files -> Reads/writes a file into/from the memory.
guid_gen -> Generates valid Age Guids
useful -> get_microseconds, get_current_time, ask, get_host_info utils
whatdoyousee -> Encrypt/Decrypt Xtea files.

**Parsers**
-----------
config_parser -> Generic configuration parser for any application. (with all the structures)
ageparser -> Parses out *.age files, also defines the structs.

**Messages**
------------
g*msg -> Generates messages
p*msg -> Parses messages

**Drivers-Subsystems**
----------------------
sql -> MySql driver

auth_db -> SQL authentication related code (kAuth)
cgas_auth -> Plasma authentication gateway (kAuth)
auth -> Authentication Driver (validates users) (kAuth)

meta_db -> Meta Server SQL related code (kMeta)

settings_db -> Configuration settings SQL related code.

vault_db -> Vault SQL code (kVault)
vaultsubsys -> Client/Server vault basic main subsystem.
vserversys -> Server vault subsystem.

lobbysubsys -> Lobby Server Subsystem.

trackingsubsys -> Tracking Server Subsystem.

gamesubsys -> Game Server Subsystem.
pythonsubsys -> Python SubSystem.

**Vault**
---------
vaultstrs -> Vault Structures
vnodes -> Parses vault nodes.
vault_obj -> Vault Objects parser, and vault packer/unpacker.
htmldumper -> Dumps vault streams into an html file
vault_advp -> Parses Vault server messages.
vault_tasks -> Vaultask parsers and generator.

**SDL**
-------
sdlstrs -> SDL Object Structures
sdl_obj -> SDL Object Parser
sdl_desc -> plain text SDL descriptor compiler/dissasembler.
sdlparser -> Binary SDL parser.
urusdl -> Testing tool, and SDL editor.

**Depreceated**
---------------
adv_gamemsgparser -> UruObjects parser (prp objects, for load clone mainly)
injector -> Loads/writes a message from the memory. (use files instead)
msg_parser -> Parses some game messages (for load clone an others)

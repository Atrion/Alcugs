
**Basic**
---------
(without auth)

*gbasicmsg -> Generates basic messages. NetMsgPing, NetMsgLeave, NetMsgTerminated, NetMsgAlive.
*pbasicmsg -> Parser for basic messages. NetMsgPing, NetMsgAlive.

**Auth**
--------

*gcauthmsg -> Generates client auth messages. NetMsgAuthenticateHello, NetMsgAuthenticateResponse.
*pcauthmsg -> Parser for client auth messages. NetMsgAuthenticateHello, NetMsgAuthenticateResponse.

*gauthmsg -> Generates server auth response messages. NetMsgAuthenticateChallenge, NetMsgAccountAuthenticated.
pauthmsg -> Parser for server auth response messages. NetMsgAuthenticateChallenge, NetMsgAccountAuthenticated.

*gscauthmsg -> Generates Auth server client messages. NetMsgCustomAuthAsk.
*pscauthmsg -> Parser for Auth server client msg. NetMsgCustomAuthAsk.

*gsauthmsg -> Generates Auth server msg. NetMsgCustomAuthResponse.
*psauthmsg -> Parses Auth server msg. NetMsgCustomAuthResponse.

**Basic**
---------
(after the auth)

*gnetmsg -> Generates basic messages before auth. NetMsgSetTimeout.
*pnetmsg -> Parses the above ones.
